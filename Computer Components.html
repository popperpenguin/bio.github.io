Inputs

Keyboard

This is one of the most important input that a computer needs. A keyboard allows people to type anything they want onto a search engine. They can search anything they want and they can use the keyboard to play games.

Ball Mouse

This is another important input. This input device is operated by rolling its ball across a flat surface. The mouse is used to control the on screen pointer by pointing, clicking, double clicking, or dragging objects on the screen.

Optical Mouse

This is another mouse that people use. This mouse is an advanced computer pointing device. It usses Light-Emitting Diode (LED) in place of the traditional mouse ball. Movement is detected by sensig changes in reflected light.

Output Devices

Monitor

A monitor is a display screen that is used to provide "output" to the user. It is where you can view the information you are working on.

Printer

This is an output device that produces a hard of the information. Information to the user that is in printed form.
